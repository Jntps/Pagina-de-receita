## Visão Geral

### O Desafio

O desafio consistiu em construir uma página de receita o mais fiel possível ao design original, garantindo a responsividade e a correta aplicação do Guia de Estilos (cores, tipografia e espaçamentos).

### Tecnologias Utilizadas

* HTML5 Semântico (uso de tags como `<main>`)
* CSS Customizado
* Conceitos de Box Model (Padding, Margin e Max-width)
* Fontes do Google Fonts (Young Serif e Outfit)
* Imagens responsivas
* Seletores avançados do CSS (Classes, Descendentes e Pseudo-elementos)

### O que eu aprendi

Durante a construção deste projeto, consolidei fundamentos cruciais para a construção de interfaces web. Como estudante de desenvolvimento de software e aplicativos mobile, entender a fundo o comportamento dos elementos na tela é essencial. 

Um dos meus maiores aprendizados foi dominar o controle de seletores no CSS para aplicar estilos com precisão cirúrgica sem poluir o HTML, além de utilizar pseudo-elementos modernos para estilizar marcadores de listas:

```css
/* Utilização do pseudo-elemento ::marker para estilizar as listas */
li::marker {
  color: hsl(14, 45%, 36%);
  font-weight: bold;
}
